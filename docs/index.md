# Qxotic Libraries

Java libraries for working with LLM model formats and inference.

| Library | Description |
|---------|-------------|
| [Toknroll](/toknroll) | Token-perfect LLM tokenization with BPE implementations |
| [Safetensors](/safetensors) | Read and write HuggingFace's Safetensors model format |
| [JSON](/json) | Strict, minimal JSON parser and printer |
| [GGUF](/gguf) | Read and write GGUF quantized model files |
